# Practice Angular Components

Make a project to practice making angular 17 components and testing them

# Installation

```sh
npm i
```

# Running

```sh
ng serve --open
```

## Running Unit Tests

```sh
ng test
```

## Running Unit Test Code Coverage

```sh
ng test --no-watch --code-coverage
```
